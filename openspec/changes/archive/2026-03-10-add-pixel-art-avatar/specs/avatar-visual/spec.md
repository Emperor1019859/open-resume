## ADDED Requirements

### Requirement: Pixel Art Avatar Component
The system SHALL display a pixel art-style avatar representing an engineer in the header section.

#### Scenario: Avatar Visibility
- **WHEN** the page is loaded
- **THEN** the pixel art avatar is visible in the header, consistent with the terminal-like aesthetic.

### Requirement: "Saying Hi" Animation
The avatar SHALL feature a "saying hi" (waving hand) CSS animation to engage viewers.

#### Scenario: Animation Execution
- **WHEN** the avatar is rendered
- **THEN** the avatar's right arm cycles through a waving motion (up and down/side to side).
