Enhance the visual appeal and interactivity of the hero section by adding a personalized greeting animation. This provides a more engaging and modern feel to the terminal-themed resume.

- **Styling**: Apply CSS to ensure the video integrates seamlessly with the existing dark terminal aesthetic and remains responsive.
