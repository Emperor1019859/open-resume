PERFORMANCE ENGINE

1000x-rps-server

四階段架構演進優化：

  • • v1 → v2: 導入 Atomic Lua Scripts。將 Redis 並發邏輯腳本化，大幅減少網路往返開銷與鎖競爭。
  • • v2 → v3: Gunicorn 多核併發。切換至 Gunicorn + 4 Uvicorn workers，極大化 CPU 多核心效能利用率。
  • • v3 → v4 (Async Kafka): 非阻塞異步發送。重構 kafka_worker 採用 asyncio.create_task 並移除 send_and_wait。
  • • 結果: 最終達成 2,977 RPS (6x 提升) 且 1000 位用戶下延遲僅 9.7 ms (縮減 99.3%)。

4-Stage Performance Evolution:

  • • v1 → v2: Atomic Lua Scripts to eliminate Redis locking and reduce RTT.
  • • v2 → v3: Gunicorn Multi-workers to maximize multi-core CPU utilization.
  • • v3 → v4 (Async Kafka): Non-blocking sends using asyncio.create_task.
  • • Result: Hit 2,977 RPS (6x boost), 99.3% latency reduction (9.7 ms).
